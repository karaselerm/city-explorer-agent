# Диаграммы системы

Диаграммы в этой папке покрывают ключевые вопросы Milestone 2:
- внешние границы и зависимости;
- контейнеры и их связи;
- компоненты ядра оркестратора;
- execution workflow с ветками ошибок;
- data flow, хранение и логирование.

| Файл | Назначение |
|---|---|
| `c4-context.md` | C4 Context: пользователь, система и внешние сервисы |
| `c4-container.md` | C4 Container: контейнеры и интеграции |
| `c4-component.md` | C4 Component: внутреннее устройство ядра |
| `workflow.md` | Пошаговое выполнение запроса + fallback |
| `data-flow.md` | Потоки данных, политика хранения и журналирования |
